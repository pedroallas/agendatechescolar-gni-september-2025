## 🚀 Quick Start

1. **Desenvolvimento Local**

   ```bash
   npm install
   npm run dev
   ```

2. **Credenciais Demo**

   - Admin: `admin@escola.edu.br` / `admin123`
   - Professor: `professor@escola.edu.br` / `user123`

3. **Seed Database**
   ```bash
   # Via API
   curl http://localhost:3000/api/seed
   ```
